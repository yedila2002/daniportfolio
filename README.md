# Yenny Daniela Velasquez Paternina - Portfolio

A professional, creative, and colorful personal portfolio for **Yenny Daniela**, a Digital Marketing & Analytics professional based in Barcelona.

## 🚀 Features

- **Modern & Responsive Design**: Custom Tailwind CSS styling with a vibrant coral, purple, and teal color palette.
- **Interactive Experience**: Smooth scroll navigation, Framer Motion animations, and hover effects.
- **Comprehensive Sections**:
  - **Hero**: Introduction and CTA for CV download.
  - **About Me**: Professional bio, languages, and hobbies.
  - **Skills**: Hard and soft skills grouped for clarity.
  - **Experience**: Timeline of professional roles across several countries.
  - **Education & Certifications**: Academic background and industry certifications.
  - **Portfolio**: Showcase of Master's projects and personal website work (Digify Active).
  - **Contact**: Direct links to email, LinkedIn, and phone.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Font**: Google Fonts (Cormorant Garamond, DM Sans)

## 📦 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / pnpm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/daniela-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
