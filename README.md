# Yenny Daniela Velasquez Paternina - Portfolio

A professional, creative, and colorful personal portfolio for **Yenny Daniela**, a Digital Marketing & Analytics professional based in Barcelona.

## 🚀 Features

- **Modern & Responsive Design**: Custom Tailwind CSS styling with a vibrant coral, purple, and teal color palette.
- **Interactive Experience**: Smooth scroll navigation, Framer Motion animations, and hover effects.
- **Comprehensive Sections**:
  - **Hero**: Introduction and CTA for CV download.
  - **About Me**: Professional bio, languages, and hobbies.
  - **Skills**: Hard and soft skills grouped for clarity.
  - **Experience**: Timeline of professional roles across several countries.
  - **Education & Certifications**: Academic background and industry certifications.
  - **Portfolio**: Showcase of Master's projects and personal website work.
  - **Contact**: Direct links to LinkedIn and contact form.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 How to Connect to GitHub & Vercel

### 1. Connect to GitHub
1. Create a new, **empty** repository on [GitHub](https://github.com/new).
2. Copy the repository URL (e.g., `https://github.com/your-username/your-repo-name.git`).
3. In your terminal (inside this folder), run:
   ```bash
   git remote add origin YOUR_REPOSITORY_URL
   git branch -M main
   git push -u origin main
   ```

### 2. Deploy to Vercel
1. Go to [Vercel](https://vercel.com/new).
2. Import your newly created GitHub repository.
3. Keep the default settings and click **Deploy**.
4. Your site will be live in minutes!

## 🛠️ Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
